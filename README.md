Proyecto PROGRAMACIÓN GYM


-Es una aplicación sobre el gestionamiento de un gimnasio , donde al ser tu el administrador puedes tanto como agregar o eliminar clientes , también puedes ver todos los clientes que hay en el gym apuntados , otras funcionalidades del administrador es poder agregar maquinas al cliente o eliminarlas 

-Tenemos también un apartado de listar máquinas , donde podremos ver los tipos de salas que hay con su código tanto de sala como de máquina junto al nombre de la máquina .

-Al ser administrador tu puedes añadir salas y máquinas , como también puedes eliminarlas.





## 🛠️ Tecnologías Utilizadas

El proyecto está desarrollado utilizando las siguientes tecnologías y herramientas del ecosistema Java:

* **Java 21 / 17:** Uso de las últimas características del lenguaje para garantizar un rendimiento óptimo.
* **JavaFX:** Para el diseño y desarrollo de una interfaz de usuario rica, moderna y nativa.
* **FXML:** Separación limpia de la vista y la lógica mediante archivos de marcado XML para los diseños de las pantallas.
* **MySQL** Base de datos relacional para la persistencia segura de los datos (Clientes, Salas, Máquinas y sus relaciones).
* **JDBC (Java Database Connectivity):** Conexión y ejecución de consultas directas, optimizadas mediante `PreparedStatement` para prevenir la inyección SQL.
* **Patrón de Diseño DAO (Data Access Object):** Separación total entre la lógica de negocio y los accesos a la base de datos para un código limpio y mantenible.
* **Maven:** Gestión de dependencias y automatización del ciclo de vida del software.
